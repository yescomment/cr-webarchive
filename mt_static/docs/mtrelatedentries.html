NAME

mt-relatedentries - a plugin for Movable Type that displays entries that are related to the current entry.

SYNOPSIS

 <MTRelatedEntries lastn="3" previous="1">
  <p><a href="<$MTEntryLink$>"><$MTEntryTitle$></a> -
  <i><$MTEntryDate format="%b %d, %Y"$></i></p>
 </MTRelatedEntries>

DESCRIPTION

Related Entries is a plugin for Movable Type that displays entries that are related to the current entry.

If you place your entries in categories, you can use this tag provide your readers with the same sort of ``Related Article'' feature that is often found on magazine and news sites.

Prior to version 3.0, the plugin was copyright 2002 Adam Kalsey, Kalsey Consulting Group.

TAGS

  • MTRelatedEntries [lastn=``'' source=``'' previous=``'']
    • This tag is a container and (essentially) specialized version of the MTEntries tag. This tag must be used in the context of another entry.

    This tag has a few optional arguments. lastn controls the number of realted entries similar to the same named argument of MTEntries. The default is 25. source must be either 'keywords' or 'primary_category'. The default is 'primary_category'. The last is previous which is boolean the denotes whether to list related entries previous to the one in context (1) rather then the most current (0, the default behavior).

PERFORMANCE CAVEAT

Using 'keywords' as your source is quite slow and resource intensive. Your rebuild will be slow, very slow. Before you implement such a scheme be sure you are willing to make this tradeoff.

INSTALLATION

To install mt-relatedentries.pl, simply place the mt-relatedentries.pl file into your MovableTypeHome/plugins/ directory. Copy the HTML documentation file in the docs subdirectory to MTStaticPath/docs directory.
