The amazing galery viewer from JonDesign's SmoothGallery for NextGen Gallery.

== Description ==

**Nextgen Smooth Gallery**: A Wordpress Plugin that allows you to use the amazing **JonDesign's Smooth Gallery** viewer on your **NextGen-Gallery** galleries.

In order to use the galery viewer from [JonDesign's SmoothGallery](http://smoothgallery.jondesign.net/) on your blog, you need to be using the best Wordpress gallery manager: [NextGen Gallery](http://wordpress.org/extend/plugins/nextgen-gallery/).

You can see the instructions and the live demo at [Uninuni - Nextgen Smooth Gallery](http://uninuni.com/wordpress-plugin-nextgen-smooth-gallery/)
